<h3 class="my-2">Statement</h3>
    <p>
        Welcome to SourceZ`s problemset! This is just a trivial problem, so that you get used to the judge.
        <br>
        Print an isosceles right triangle with side length equal to N, using only asterisks.
    </p>

    <hr>

    <h3 class="my-2">Input format</h3>
    <p>
        File welcome.in contains, on the first line, one positive integer, N.
    </p>

    <hr>

    <h3 class="my-2">Output format</h3>
    <p>
        File welcome.out should contain N lines.
        <br>
        Line i should contain i '*' symbols (without quotation marks), not separated by spaces.
    </p>

    <hr>

    <h3 class="my-2">Restrictions</h3>
    <p>
        1 <= N <= 100
    </p>

    <hr>
